## Project Overview
This project was created as part of Task 2 of the Future Interns UI/UX Internship.
The goal of this task is to design a mobile-first UI for a mental health habit tracking app that helps users build positive daily habits in a simple, calm, and supportive way.

The design focuses on empathy-driven UX, clarity, and ease of use rather than complex features.

## App Concept
The app helps users track daily mental wellness habits such as:
- Meditation
- Journaling
- Walking or light activity

The interface is designed to feel minimal, calming, and encouraging.

## Screens Included
- Home Screen
- Habit Dashboard Screen
- Activity Details Screen

## Tools Used
- Canva (UI Design and Prototype)

## Design Notes
- Mobile-first design approach
- Soft colors and clean layout for mental wellness
- Simple user flow for better accessibility
- Screens are exported as PNG files and uploaded for documentation
